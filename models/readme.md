## 📦 Model Files

Due to size and licensing restrictions, the model files are not included in this repository.

Please download the required models from the following sources:

* Face Detection Model: [[Link](https://aihub.qualcomm.com/compute/models/face_det_lite?domain=Computer+Vision&useCase=Object+Detection)]
* Face Recognition Model: [[Link](https://github.com/yakhyo/face-reidentification/releases/download/v0.0.1/w600k_mbf.onnx)]
* Liveness Detection Model: [[Link](https://github.com/facenox/face-antispoof-onnx/blob/e3d59c8cbec4c9c5e939be225ece63b43e1f93e4/models/best/98.20/best_model_quantized.onnx)]

After downloading, place them in the `models/` folder.
